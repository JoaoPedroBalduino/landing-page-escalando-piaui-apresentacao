## 📋 Sobre o Projeto

Projeto desenvolvido para a **Escalando Piauí**, empresa especializada em **Alpinismo Industrial** — serviços de manutenção em altura para fachadas de prédios, limpeza de vidros, troca de pastilhas e fossos de elevador.

O objetivo foi criar uma presença digital profissional e de alta conversão, transmitindo **segurança, autoridade e modernidade** — valores essenciais para um público B2B exigente como síndicos, administradoras e engenheiros.

## ✨ Destaques Técnicos

| Recurso | Descrição |
|---|---|
| 🎨 **Design** | Dark theme com paleta industrial — preto, amarelo-terra e laranja |
| 📱 **Responsividade** | Layout otimizado para mobile, tablet e desktop |

## 🚀 Como Visualizar Localmente

Clone o repositório e abra com qualquer servidor local:

```bash
# Clone o projeto
git clone https://github.com/seu-usuario/escalando-piaui.git
cd escalando-piaui

# Opção 1 — Python (sem instalação)
python3 -m http.server 8080

# Opção 2 — Node.js (com npx)
npx serve .

# Opção 3 — VS Code
# Instale a extensão "Live Server" e clique em "Go Live"
```

Acesse em: **`http://localhost:8080`**

## 🌐 Deploy no Netlify

O projeto está publicado e configurado com deploy contínuo via Git.

| Configuração | Valor |
|---|---|
| **URL** | [escalando-piaui.netlify.app](https://escalando-piaui.netlify.app/) |
| **Build command** | _(vazio — HTML estático)_ |
| **Publish directory** | `.` |
| **Deploy automático** | ✅ A cada push na branch `main` |

Para publicar sua própria versão:

1. Faça um fork deste repositório
2. Acesse [netlify.com](https://netlify.com) e conecte o repositório
3. Configure publish directory como `.` e clique em **Deploy**

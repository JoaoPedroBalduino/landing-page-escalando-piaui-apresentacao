# Escalando Piauí — Landing Page

Landing page institucional desenvolvida para apresentar os serviços da **Escalando Piauí**, com foco em conversão via WhatsApp e presença digital profissional.

## ✨ Destaques do projeto

- Design moderno e responsivo
- Estrutura otimizada para apresentação comercial
- CTA direto para orçamento via WhatsApp
- Bloco de contato com telefone, Instagram e e-mail

## 🧱 Estrutura

```text
.
├── index.html
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
└── netlify.toml
```

## 🚀 Como visualizar localmente

Abra o `index.html` no navegador  
ou rode um servidor local simples:

```bash
python3 -m http.server 8080
```

Depois acesse: `http://localhost:8080`
